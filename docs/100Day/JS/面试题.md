随机抽奖程序 接受一个数组 arr, n ,从数组中抽出n个人
```js
function foo(arr, n) {
    const result = [];
    let i = 0;
    while( i < n) {
        const value = arr.splice(Math.floor(Math.random() * arr.length), 1)
        result.push(value[0]);
        i++;
    }
    return result;
}
console.log(foo([1, 2, 3, 6, 9, 10], 2));
```
